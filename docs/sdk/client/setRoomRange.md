### setRoomRange(int $min, int $max)

Définit une fourchette de nombre de pièces.

#### Paramètres

| Param | Type | Description |
| --- | --- | --- |
| $min | int | Nombre de pièces minimum désiré |
| $max | int | Nombre de pièces maximum désiré |

#### Exemple 

```php
$Client->setRoomRange(3, 6);
```
